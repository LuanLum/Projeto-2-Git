Projeto 1 — Navegador como IDE

• Implemente o FizzBuzz de 1 a 100 no console do navegador.
• Implemente a sequência de Fibonacci em estilo funcional puro (sem for, let, var, while ou mutação de estado).
• Demonstre que entende a diferença entre let e const criando exemplos que mostram onde cada um falha ou funciona.
• Crie uma class com um método simples (speak()) e instancie um objeto dela.
• Crie e ordene um array de strings com Array.sort(), sem usar .localeCompare().
• Crie um Map com três pares chave-valor e percorra com for...of.
• Use fetch() para consumir uma API pública (https://jsonplaceholder.typicode.com/todos/1) e imprima o JSON no console.

Projeto 2 — Git

• Instale o Git e configure com git config (user.name e user.email).
• Crie um repositório local com git init, adicione o código do projeto anterior.
• Crie uma conta no GitHub e configure chave SSH (ssh-keygen).
• Crie um repositório remoto e suba o repositório local usando ssh.
• Demonstre que sabe usar .gitignore.
• Demonstre que conhece o padrão convencional de mensagens de commits(feat:, fix: etc).
